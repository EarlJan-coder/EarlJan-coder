## Featured Projects

### Blue Collar Inventory Management System  
A full-stack inventory management system designed for blue-collar businesses such as automotive and motorcycle parts shops. It includes item management, categories, suppliers, purchase orders, sales tracking, and user-based inventory features.

**Tech Used:** Next.js, React, Tailwind CSS, Node.js, Express.js, PostgreSQL, Neon, Drizzle ORM

---

### ReviewlyAI  
An upcoming AI-powered reviewer web application designed to help students generate reviewers, quizzes, summaries, and study materials from uploaded learning resources.

**Focus:** SaaS Development, AI Integration, LLMs, Student Productivity

---

### LANHS Document Request Management System  
A document request and management system for school records such as certificates, student documents, and request tracking. The system includes role-based access for Admin, Registrar, and Student/Alumni users.

**Planned Features:** XLSX grade import, PDF certificate export, QR code verification, email sending, and blockchain-based transaction logging.

---

### Bookified  
An AI-driven book reader concept that helps users interact with books, generate reviews, and improve reading comprehension through AI-powered features.
